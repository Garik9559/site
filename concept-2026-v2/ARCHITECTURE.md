## Ключевые страницы и функционал

### 1. Каталог продукции `/catalog/`
- Фильтрация: по категории, материалу, цене, размеру
- Поиск с автоподсказками
- Карточка товара:
  - Фото/3D-превью (можно крутить)
  - **Переключатель материала** (архимрамор / архикамень / пенополистирол) — при смене:
    - Меняется цена
    - Меняется вес
    - Меняется остаток на складе
    - Меняется фото (если есть различия)
  - Размеры, чертёж сечения
  - **Скачать**: DWG, PDF чертёж, STL/3DS для 3D-программ, инструкция по монтажу
  - Кнопка "В корзину" / "В лист заказа"
  - Похожие товары
  - Применённые проекты (где использовался этот элемент)
- SEO: уникальные title/description для каждой карточки, Schema.org Product разметка

### 2. Корзина / Лист заказа `/cart/`
- Список выбранных позиций с количеством
- Итоговая стоимость и вес
- Кнопка "Оформить заявку" → форма с контактами → заявка в CRM
- В будущем: онлайн-оплата (Юkassa / Тинькофф)
- Экспорт списка в PDF

### 3. Калькулятор `/calculator/`
- Пошаговый визард:
  1. Тип здания (дом, коммерческое)
  2. Количество этажей
  3. Периметр / площадь фасада
  4. Выбор элементов (карнизы, наличники, и т.д.)
  5. Выбор материала
  6. Расчёт → PDF-смета на email

### 4. Блог `/blog/`
- Категории: Фасадный декор, Дизайн, Монтаж, Материалы, Новости
- Статья: содержание (TOC), автор, дата, время чтения, теги
- Комментарии (WordPress native)
- Связанные товары из каталога внутри статьи
- FAQ-блоки в статьях (Schema.org FAQPage)

### 5. Дилерская сеть `/dealers/`
- Интерактивная карта России с маркерами
- Список регионов → города → карточка дилера:
  - Название компании
  - Адрес, телефон, email
  - Ссылка на сайт
  - Фото шоурума
  - Рейтинг

### 6. Стать дилером `/become-dealer/`
- Условия сотрудничества (оптовые цены, маркетинговая поддержка, обучение)
- Многошаговая форма заявки:
  1. Контактные данные
  2. Информация о компании (ИНН, регион, опыт)
  3. Тип сотрудничества
  4. Загрузка документов

## Техническая SEO-архитектура (2026)

### Schema.org разметка
- `Organization` — на всех страницах
- `Product` — карточки товаров (цена, наличие, материал, рейтинг)
- `FAQPage` — FAQ и статьи с вопросами
- `Article` / `BlogPosting` — блог
- `BreadcrumbList` — хлебные крошки
- `LocalBusiness` — контакты и дилеры
- `HowTo` — инструкции по монтажу
- `Service` — услуги

### AI Search Optimization (2026)
- `llms.txt` и `llms-full.txt` в корне сайта — структурированная информация для LLM-поисковиков
- Open Graph + Twitter Cards для всех страниц
- JSON-LD разметка встроена в каждую страницу
- Semantic HTML5 (article, section, nav, main, aside)
- Длинные экспертные статьи (2000+ слов) для E-E-A-T
- Внутренняя перелинковка: товар ↔ статья ↔ проект ↔ материал

### Скорость
- Static Site Generation для каталога (ISR)
- Изображения: WebP/AVIF с srcset, lazy loading
- Critical CSS inline, остальной CSS — async
- JS: code splitting, lazy load тяжёлых компонентов (карта, 3D-превью, калькулятор)
- CDN для статики

## Рекомендуемый стек

### Вариант A: WordPress + WooCommerce (эволюционный)
**Для:** если нужно быстро и в рамках текущей инфраструктуры

| Компонент | Решение |
|---|---|
| CMS | WordPress 6.x |
| Каталог | WooCommerce + ACF Pro (кастомные поля: материал, вес, остаток) |
| Тема | Кастомная на базе Tailwind CSS + Alpine.js |
| Переключатель материала | AJAX + ACF repeater (вариации по материалу) |
| Корзина | WooCommerce cart (режим "запрос", без оплаты) |
| Калькулятор | Custom plugin + React widget |
| Блог | WordPress native |
| SEO | Rank Math Pro + Custom Schema |
| Хостинг | VPS (Timeweb Cloud / Beget) + Redis + CDN |
| Поиск | SearchWP или Elasticsearch |

**Плюсы:** быстрый запуск, знакомая CMS, много готовых решений
**Минусы:** производительность при большом каталоге, сложность кастомизации WooCommerce

### Вариант B: Headless (Next.js + WordPress API) — рекомендую
**Для:** максимальная скорость, гибкость, масштабируемость

| Компонент | Решение |
|---|---|
| Frontend | Next.js 15 (App Router, Server Components) |
| CMS / Backend | WordPress как Headless CMS (REST API / WPGraphQL) |
| Каталог | Custom post types + ACF Pro → GraphQL |
| Переключатель материала | React-компонент, данные из вариаций |
| Корзина | Zustand store → API endpoint → CRM |
| Калькулятор | React multi-step form |
| Стили | Tailwind CSS 4 |
| Анимации | Framer Motion |
| 3D-превью | Three.js / React Three Fiber (для STL-моделей) |
| Карта дилеров | Yandex Maps API 3.0 + кластеризация |
| Поиск | Meilisearch (мгновенный полнотекстовый) |
| SEO | Next.js Metadata API + JSON-LD |
| Хостинг frontend | Vercel / VPS с Docker |
| Хостинг backend | Текущий Timeweb (WordPress API) |
| CDN | Cloudflare |
| Аналитика | Yandex.Metrica + Plausible (privacy-friendly) |

**Плюсы:**
- PageSpeed 95-100 из коробки (SSG/ISR)
- Мгновенные переходы между страницами (SPA)
- Полный контроль над UI и UX
- WordPress остаётся для контент-менеджеров (привычный интерфейс)
- Масштабируется на 10,000+ товаров

**Минусы:**
- Нужен фронтенд-разработчик (React/Next.js)
- Первоначальная настройка сложнее
- Два хостинга (frontend + backend)

### Мой выбор: Вариант B (Headless)
Причина: для каталога с переключателем материалов, 3D-превью, калькулятором и корзиной — чистый WordPress будет тормозить. Next.js даст PageSpeed 95+ и UX на уровне лучших e-commerce сайтов.

## Дизайн-система

### Палитра
- **Primary:** #0f1923 (глубокий антрацит)
- **Accent:** #c9a96e (тёплое золото)
- **Surface:** #f7f5f2 (тёплый крем)
- **Text:** #1a1a2a
- **Muted:** #6b7280
- **Success:** #16a34a
- **Error:** #dc2626

### Типографика
- Заголовки: **Cormorant Garamond** (классика, архитектура)
- Текст: **Inter** (читаемость, нейтральность)
- Моноширинный (артикулы): **JetBrains Mono**

### Принципы
- **Много воздуха** — просторные отступы, элементам не тесно
- **Фото > текст** — крупные фото проектов и продукции, текст как поддержка
- **Микровзаимодействия** — hover-эффекты, плавные переходы, skeleton-загрузки
- **Тёмный header + светлый контент** — премиальное восприятие
- **Mobile-first** — 70% трафика строительных сайтов с мобильных
